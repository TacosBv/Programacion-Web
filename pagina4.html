Proyecto Web de Gustavo Britez

Objetivos del Proyecto

El objetivo principal de este proyecto es crear una serie de páginas web que muestren diferentes aspectos de mi perfil personal, habilidades, intereses y conocimientos. A través de estas páginas, se busca demostrar mi capacidad para desarrollar sitios web de manera estructurada y visualmente clara. Además, el proyecto busca profundizar en la comprensión de HTML, crear contenido visual atractivo, y fomentar el uso de buenas prácticas en el desarrollo web.

Tecnologías Usadas

  • HTML5
  • CSS (opcional, para diseño futuro)
  • JavaScript (opcional, para interactividad futura)
  • Editor de código: Visual Studio Code
  • Git y GitHub para control de versiones
